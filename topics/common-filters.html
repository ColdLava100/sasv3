🔄 Common Filters — Shared Across Objects ▶

From course: "Common filters are tied to a data source, are able to be shared between objects, are available to the entire report, and, if edits are made, those changes are applied everywhere the filter is used."

🔄 What is a Common Filter?

A common filter is a filter that is shared between objects using the same data source. It can be a basic, advanced, or post-aggregate filter that you convert to a common filter.

"A common filter can be a basic, advanced, or post-aggregate filter that is shared between objects."

📋 How to Create One

Step 1: Create a filter (basic, advanced, or post-aggregate)
Step 2: Click More on the filter
Step 3: Select Change to common filter
Step 4: Apply to any object using the same data source

The common filter appears in the Data pane under the Common Filter group.

🔑 The 4 Key Attributes of Common Filters

🔗 Tied to a data source

Each common filter belongs to a specific data source. Only objects using that data source can use the filter.

🤝 Shared between objects

Apply the same filter to multiple objects — bar charts, tables, KPIs — all stay in sync.

📄 Available to entire report

✅ Across pages too! The filter works on every page of the report, not just the page where it was created.

⚡ Edits apply everywhere

"If the common filter is edited, those edits are immediately applied to all the objects using the common filter."

🎯 Exam Question

Question: "Common filters are tied to a data source and can be shared between objects. What are two additional attributes of common filters?"

✅ A) Available to the entire report. — "Available to the entire report (yes, across pages)."
✅ D) Edits are applied instantaneously everywhere the filter is used. — "Edits are immediately applied to all objects using the filter."
❌ B) Available only on the page the filter is created. — WRONG! It's available across ALL pages.
❌ C) Administrators have to publish common filters. — WRONG! Any user can create and apply common filters.

📋 Common Filter vs Object-Specific Filter

Aspect🔗 Common Filter🔒 Object-Specific Filter
ScopeEntire report (all pages)Single object only
EditsApply to all objects using itOnly affect that one object
ReusabilityCan be applied to any object with same data sourceCannot be reused
Appears inData pane → Common Filter groupOnly in object's Filters pane
Revert?Can revert to object-specific for one-off changesCannot be promoted to common
🔧 Reverting a Common Filter: "For a specific instance of a common filter, you can revert the common filter to an object-specific copy." This lets you make one-off changes to a single object without affecting others.

Common = Cross-page, Connected, Changes propagate
