🎨 5.1 Display Rules — All 7 Types + Levels

"Display rules dynamically color category and measure values to direct attention to specific results."

7 Rule Types: Expression (Boolean, accepts numeric params), Color-Mapped Values (category value→color), Gauge (measure interval), Icon (value range→icon), Bullet (performance vs target), Slider (value selection), Thermometer (thermometer indicator)

Levels display rules can be applied:
• Report-level: Applies to all objects. Add via + button. Edit to add more color-mapped values. Considerations: objects may need Group role, may need object-level rule for enhancement, may not be affected.
• Table-level (List table): Expression, Color-Mapped, Gauge, Icon, Bullet. Crosstab: Expression only — requires measure in Data Role, can be layered (first rule has priority), applied to column or intersection.
• Graph-level: Color-Mapped Values + Expression. Style: graph element or background. Geo map: Icons only for Geo Coordinate/Geo Network. Basic icons (respond to color) + Custom icons (static image). Display Rule OVERRIDES Color Data Role.
• Gauge-level: Gauge type only. Default 3-interval display rule defined. Auto-calculate interval endpoints.
• Trigger Alerts: Expression type. Enable/disable from edit mode. Subscribers get email + notification. Recipients must be registered SAS Viya users.

🔢 5.2 Parameters — All 4 Types

"A parameter is a variable whose value can be changed and referenced by other objects. Adds interactivity."

1. Numeric Parameter: Viewer enters a number. Used with slider control. Create via New data item, Expression Editor, or right-click. Specify format + starting value.
2. Character Parameter: Viewer chooses a category or enters alphanumeric string. Used with dropdown/list/text input.
3. Date Parameter: Viewer selects a date or datetime. Used with date picker or slider.
4. Expression-Based Parameter: Value set by formula — NOT affected by filters or actions. Based on entire data source. Value remains constant regardless of user selections. Updated when data refreshes. Useful for benchmarks, percent of total calculations.

Use parameters in: Calculations, Filters, Display rules, Ranks, URLs (interactive links).
Create: New data item button, Expression Editor, right-click in Data pane.

🔗 5.3 Links — Interactive URLs

4-step process for interactive links: Research → Hardcode → Parameterize → Test
URL structure types: Path, Query, or File.
Data source mapping: When source and target use different data sources → prompted to map. Global map data available for page links. Mapped columns must have same format. Column can be mapped only once.
Considerations: Filter applied to prompt areas and canvas (default). Alternative: Use "Set prompt bar values" option.
