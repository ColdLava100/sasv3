🎨 SAS Graph Builder — Introduction

"If the out-of-the-box objects in Visual Analytics do not meet your needs: SAS Graph Builder enables you to create your own custom graph and map templates."

Access: From the Applications menu (available to all users)
Two ways to create:
• From scratch: Select graph elements → create layout → configure roles → adjust options → save and use
• Graph Gallery: Start from pre-built template — all gallery options are Visual Analytics objects

Workflow: Drag and drop individual graph elements onto a layout grid. Each element placed in a cell. Default: everything starts in Cell A1. Configure available data roles, specify visual elements and options. Save → custom template appears in Objects pane for use in reports.

🧩 Graph Elements & Layout

Over 15 graph elements available: Bar, Line, Scatter, Pie, etc.

Layout types:
• Overlay: One graph element superimposed on another in the same cell. NOT all elements compatible (e.g., pie + bar won't work). A Shared role is added automatically.
• User-defined lattice: Multiple cells arranged in rows and columns. Each cell is independent with its own graph element. Cells auto-sized but can be manually adjusted.

Role mapping: When you drop a graph element onto the layout, roles appear in the right pane. Assign data items to roles to define the graph.

💾 Save, Export & Use

Save template: Saved to SAS Content. Appears in Objects pane under graph gallery section.
Use in report: Drag from Objects pane like any standard VA object.
Export: Can export template JSON to share across environments.
Useful when you need: specific combination of chart types, custom layout, or unique visualization not available out-of-the-box.
