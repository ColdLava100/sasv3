🔀 Path Analysis — Link Color Options ▶

Path Analysis (also called a Sankey diagram) displays flows of data from one event to another. From course: "The path analysis object displays flows of data."

🔍 Path Analysis Anatomy

Nodes — Events/stages (vertical bars)
Links — Paths/connections between events (curved bands)
Link width — How many people/items take that path
Link color — How paths are grouped and colored

📊 All 4 Link Color Options — Same Data, Different Coloring

🎯 Your Exam Question

Question: "Which Link color option has been set on the path analysis object?"

❌ Your answer: Drop off
✅ Correct: Path
Why? Each path has its own distinct color from start to end. Path 1 = blue, Path 2 = green, Path 3 = yellow, Path 4 = purple. This is exactly what "Path" link coloring does — "Displays each path as a separate group with a distinct color."

Why not Drop off? Drop off would only have 2 colors (continuing vs drop-off). But in the diagram, EVERY path has its own unique color.

🧠 Memory Trick

Path = Personal color per journey 🎨
Drop off = Dead end vs alive 🔴🟢
Event = Every stage same color per type 🏷️
Sequence = Step number color 🔢

"Each Path gets its own Paint job."
