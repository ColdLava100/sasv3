🔗 4.1 Data Joins

"Data source joins enable you to combine two data sources. The join creates a temporary table that is executed every time the report is opened."

AspectDetails
JoinCombines two tables by matching rows and adding columns
AppendCombines two tables by adding rows with matching columns
Available in VA?✅ Joins (Left, Right, Inner, Full). ❌ Append NOT available
TemporaryStored in CASUSER caslib — visible only to creator. Cleaned up when no longer needed.
Considerations⚠️ Performance: Join runs every time report opens for every user.
⚠️ Can join only 2 tables at a time. No self-joins.
⚠️ Only simple matching. Dates must have matching formats.
⚠️ Temporary results can be very large — impacts concurrent users.

📊 4.2 Aggregated Data Sources

Creates a new temporary data source with aggregated subset of original columns. Improves performance. Can be joined back to original data source.
Used for nested aggregated measures — aggregated functions do NOT accept aggregated measures, so use data items from the aggregated data source instead.

📝 4.3-4.5 Expression Editor Functions

Conditional Logic: IF...ELSE (IF condition THEN result ELSE alt END). SELECT...WHEN...OTHERWISE for multiple conditions.

Text Functions (Simple & Advanced): Substr, Trim, Upcase, Lowcase, ReplaceWord, GetWord, Length, Cat (concatenate), Cats (concatenate without spaces)

Date & Time Functions: Year, Month, Day, Weekday, DatePart, TimePart, Today(), Now(), IntNX(interval, start, inc) — advance date, IntCK(interval, start, end) — count intervals, DHMS(date, hr, min, sec) — create datetime

Aggregated — Simple: SUM, AVG, MIN, MAX, COUNT, STD, VAR, NMiss
Aggregated — Periodic: CumulativePeriod, ParallelPeriod, Period, PeriodWithDate, RelativePeriod
Aggregated — Advanced: CoefVar, CSS, Kurtosis, PvalT, Skewness, First, Last, Suppress (hides values for privacy)
Tabular: AggregateCells (specific cells), AggregateTable (independent aggregations), Moving sum/avg/median, Cumulative functions

🎯 Key Exam Points — M2 L4:
• Join = adds columns. Append = adds rows. Append NOT in VA ⚠️
• Only simple matching available. Dates must match formats ⚠️
• Join runs every time report opens for every user — performance impact ⚠️
• IntNX = advance date. IntCK = count intervals ✅
• Aggregated functions do NOT accept aggregated measures ❌
• Suppress = obscure data for privacy. CoefVar, Kurtosis, Skewness = advanced stats ✅
• Periodic functions: Cumulative, Parallel, Period, PeriodWithDate, RelativePeriod ✅
