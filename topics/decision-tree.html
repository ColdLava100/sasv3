🌳 Decision Tree — Analytic Object ▶

Decision Tree is an analytic object in SAS VA. From course: "A decision tree is an explainable and powerful machine learning algorithm that can be used for both classification and regression tasks."

🌳 What is a Decision Tree?

How it works

"By learning from the data, a decision tree iteratively creates a set of decision rules to accurately predict the response variable."

1️⃣ Start — Pick a response variable to predict
2️⃣ Split — Find the best predictor to split on
3️⃣ Branch — Create branches based on split rules
4️⃣ Repeat — Continue splitting until done
5️⃣ Predict — New data follows rules to get prediction

White Box Model

"Decision trees are considered white box models because their structure is easily visualized, the decision-making process is transparent and easily understandable."

Unlike "black box" models (neural networks), you can SEE and EXPLAIN exactly why a decision tree made its prediction!

License Notes

Base decision tree — Included with SAS VA license ✅
Advanced decision tree — Requires SAS Visual Statistics license ⚠️

Advanced features: predictor importance, assessment plots, interactive mode (prune, edit split, train), score code generation, Model Studio integration

⚙️ Key Options

Event Level

For category responses — choose which outcome level to model. Affects predictions and False Positive Rate.

Growth Strategy

Default or Custom. Custom lets you control: max branches, max levels, min leaf size, predictor bins, pruning, bin method, reuse predictors.

Prediction Cutoff

Probability threshold — above this, the outcome is considered to occur.

Interactive Mode

Right-click to enter. Prune nodes, edit split rules, or grow tree further.

📊 Results You Can View

When you maximize the decision tree, you can see:
✅ Decision Tree — Visual tree with branches and leaves
✅ Icicle Plot — Alternative view of the tree (selected nodes highlight in BOTH views)
✅ Assessment Statistic — Shown at the top (e.g., accuracy, ASE)
✅ Predictor Importance — Which variables matter most (advanced license)
✅ Assessment Plots — Evaluate model fit (e.g., ROC, lift) — right-click to toggle
✅ Confusion Matrix — Shows correct vs incorrect predictions
✅ Score Code — Right-click to generate code for scoring new data

🎯 Exam Question Answer:
Q: "Which statement describes how a decision tree is used?"
✅ B) It segments data into branches based on input variables to predict a target outcome.

❌ A) Histograms & box plots = descriptive statistics, not decision trees
❌ C) Form-filling prediction = not what decision trees do
❌ D) Scatter plot correlation = correlation analysis, not decision trees
🧠 Memory Trick:
Decision Tree = Data Through Decision rules → Predict
"A tree doesn't describe or correlate — it splits and predicts."
