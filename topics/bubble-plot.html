🫧 Bubble Plot — Roles Explained CHARTS ▶

A bubble plot is like a scatter plot with 3 dimensions: X position, Y position, and bubble size as the third measure. Colors group by category.

📋 Chart Data Items

Data ItemChart ElementRole in SAS
Product Material CostX-axis ←→Measures (X axis)
RevenueY-axis ↑↓Measures (Y axis)
ProfitBubble Size ⭕Measures (Size)
Product LineBubble Color 🎨Group ✅

🎯 Answer Options

❌ a. Revenue → Lattice rows
Revenue = Y-axis = Measures
❌ b. Profit → Animation
Profit = Size = Measures
✅ c. Product Line → Group
Product Line = colors = Group role ✅
❌ d. Product Material Cost → Measures
It IS a measure, but Group is the specific correct role here

⚙️ Bubble Plot Roles (from course)

"Bubble plots require three measures: X axis, Y axis, and Size. Additionally, bubble plots can contain groups, accept the Color role, use lattice columns and rows, specify data tip values, display animation, and support the hidden data role."

Required:
• X axis measure
• Y axis measure
• Size measure
Optional:
• Group (color 🎨)
• Color • Lattice cols/rows
Also available:
• Animation ⏱️
• Data tip values • Hidden
🧠 Memory Trick:
• X → Product Material Cost ← Measure
• Y → Revenue ← Measure
• Size → Profit ← Measure
• Color → Product Line ← Group 🎨

"3 Measures make the bubble. The Group gives it color."
