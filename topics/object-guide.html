At a glance: which roles belong to which objects.

Object TypeColumns / RowsLattice Cols / Rows
📋 Crosstab✅ Yes❌ No
📈 Time Series Plot❌ No✅ Yes
🔵 Bubble Plot❌ No✅ Yes
📊 Bar Chart❌ No✅ Yes
🔴 Scatter Plot❌ No✅ Yes
📄 List Table❌ No (display setting only)❌ No
💡 Quick Rule: If it summarizes data with two header dimensions → Crosstab (uses Columns/Roles roles).
If it plots data on axes → Chart (uses Lattice Columns/Rows to create panels).
