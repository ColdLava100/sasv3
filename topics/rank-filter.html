🏆 Rank vs Filter CORE CONCEPT ▶

Both reduce the data you see — but they work completely differently.

🔑 The One-Sentence Difference:
🟢 Filter asks: "Does each row match my condition?" (Yes→keep, No→remove)
🟡 Rank asks: "Which rows are the Top/Bottom N by this measure?"

🧠 Memory Trick

Sort = Shuffle everything in order
Filter = Forget what doesn't match a condition
Rank = Reduce to the best/worst by a measure

"In a race, you RANK the top 3 finishers — you don't filter them."

📝 Quiz — Rank vs Filter

Q1: Show products where Sales > $10,000. Use Filter or Rank?

✅ Filter — Condition: Sales > $10,000. Rows not matching are removed.

Q2: Show Top 10 best-selling products. Use Filter or Rank?

✅ Rank — Top N by a measure = Rank.

Q3: Top 5 products by Profit, only for "Sports" category. What do you need?

✅ Both! Filter non-Sports first, then Rank Top 5 by Profit.

Q4: Rank's "Count" can be controlled by what?

✅ Parameter — Slider + parameter controls rank count dynamically.
