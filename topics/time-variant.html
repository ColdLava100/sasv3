1️⃣ Step Plot

"Useful for showing changes that occur at irregular, discrete intervals."

📏 Horizontal + vertical line segments = steps
🔍 Quickly spot exact X-axis point when Y changes
📊 Example: Interest rates over time
⚠️ Sorted ascending by X axis by default
✅ Can use: Group, Label, Lattice, Hidden

2️⃣ Vector Plot

"Useful for showing relative performance between two sets of measures from one point in time to another."

🏹 Shows direction + magnitude as arrows
📊 Example: Product quality start vs end of year
⚠️ 4 measures required: X Origin, X, Y Origin, Y
⚠️ Hide axis labels — use title to describe axes
✅ Can use: Color, Arrowheads, Group, Lattice

3️⃣ Bubble Change Plot

"Useful for showing relative performance between THREE sets of measures from one point in time to another."

🫧 Arrows + bubble sizes change
📊 Example: Market share + revenue over time
⚠️ 6 measures required!
⚠️ X/Y start/end + Size start/end
✅ Can use: Group, Bubble size range, Arrowheads

4️⃣ Waterfall Chart

"Useful for showing how incremental changes lead to a final value. Sometimes called a progressive bar chart."

🧱 Floating bars show cumulative effect
📊 Example: Revenue breakdown by quarter
🔵 Green = increase, 🔴 Red = decrease
🏁 Initial bar + Final (cumulative) bar
✅ Can use: Trend line, Color by direction, Lattice

5️⃣ Schedule Chart (Gantt)

"Useful for project management — provides a detailed overview of required tasks, duration, and order."

📋 Horizontal bars show start → finish
📊 Examples: Project tracking, holidays, work schedules
⚠️ No measure needed! Just Task + Start + Finish
✅ Can use: Group, Label per bar, Lattice

🎯 Key Exam Points:
• All 5 are in the Graphs group ✅
• Step Plot → sorted ascending by X axis ⚠️
• Vector Plot → 4 measures (X Origin+X, Y Origin+Y) ⚠️
• Bubble Change → 6 measures (the most!) ⚠️
• Waterfall → Initial bar + Final cumulative bar + Color by direction ✅
• Schedule → Task + Start + Finish (no measures!) ⚠️
• Waterfall is also called: Progressive bar, Flying bricks, Mario, Bridge 🧱
🧠 Memory Trick:
Step = Staircase (irregular intervals)
Vector = Velocity arrows (4 measures)
Bubble = Both size + direction (6 measures)
Waterfall = Water flows down incrementally
Schedule = Start + Finish (Gantt chart)
