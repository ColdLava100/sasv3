📄 List Table vs Crosstab OBJECTS ▶

Both display data in a grid — but one is flat, the other summarizes.

📄 List Table

Like an Excel sheet. Every row = one record. No summarizing.

ProductSalesQuarter
Sports$100Q1
Sports$150Q2
Toys$80Q1
"Columns" = display setting. NOT a data role.

📋 Crosstab

Like a Pivot Table. Groups + summarizes. Two header dimensions.

Q1Q2
Sports$250$150
Toys$80$120
"Columns" = a DATA ROLE that creates headers.
💡 The "Columns" Trap: Both have "columns" but different meanings!
List Table: "Columns" = display setting (which fields to show)
Crosstab: "Columns" = a Data Role (category → column headers)
