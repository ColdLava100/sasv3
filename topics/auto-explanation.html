🤖 Automated Explanation & Automated Prediction ▶

Two powerful analytic objects that automatically build models to explain or predict a response variable.

🔍 Automated Explanation

"Determines the most important underlying factors that explain a particular variable."

✅ Shows characteristics of the response variable
✅ Displays relative importance bar chart of factors
✅ Click a factor → view relationship plot
✅ Right-click a factor → remove it
✅ Natural language explanation

Roles: Response, Time (optional), Underlying factors

🎯 Automated Prediction

"Allows you to predict values of a selected response variable by specifying values for the underlying factors."

✅ Predicts future/unknown values
✅ Shows values input form for factors
✅ Runs multiple models, chooses champion
✅ Category response: Logistic regression, Gradient boosting, Decision tree
✅ Measure response: Linear regression, Gradient boosting, Decision tree

Roles: Response, Underlying factors

🔍 Automated Explanation — 3 Components

📝 1. Natural Language Summary

"Current is more common at 80.05%... The three most related factors are Debt to Income Ratio, Number of Delinquent Credit Lines..."

📊 2. Relative Importance Chart

Horizontal bar chart showing factors in descending order of importance.
✅ Click a factor → view relationship plot
❌ Right-click → remove factor

📈 3. Relationship Plot

Type depends on variable types:
📊 Measure + Measure = Scatter/heat map
📊 Category + Measure = Histogram
📊 Category + Category = Stacked bar
🗺️ Geography + Any = Geo map

🎯 Automated Prediction — How It Works

1. Select Response — Choose the variable to predict
2. Auto Assign Factors — Remaining data items become underlying factors
3. Run Models — SAS tests multiple models:
   • Category response: Logistic regression, Gradient boosting, Decision tree
   • Measure response: Linear regression, Gradient boosting, Decision tree
4. Pick Champion — Best model is selected automatically:
   • Category: Highest accuracy (lowest misclassification rate)
   • Measure: Lowest Average Squared Error (ASE)
5. Input Values — Viewer can enter values for underlying factors to see predictions

📋 Underlying Factors — What's Included vs Excluded

✅ Included as Underlying Factor❌ NOT Included
CategoriesAggregated measures
MeasuresDate and time values
GeographiesCalculated items
—Custom categories
—Derived items
—Automatically computed values (Frequency, etc.)
—Hierarchies, filters, interaction effects
⚠️ Limitations:
• Not added as source/target for actions ❌
• Excluded from report/page prompts ❌
• No manual model selection — SAS chooses champion ❌
• Use SAS Visual Statistics for more control ❌
