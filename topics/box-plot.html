• IQR = $9
Understanding outliers:
• Normal range = 1.5×IQR = $13.50 from box
• $500 and $1,200 are OUTLIERS

⚙️ The 3 Outlier Settings (from SAS course)

"Outliers can be ignored (excluded from the plot), shown (separate from whiskers), or hidden (included within the whiskers)."

🟢 SHOW

Whiskers stop at 1.5×IQR. Extreme values appear as ● dots or gray outlier bins.

🟡 HIDE (default)

Whiskers extend to include outliers. No dots shown — outliers absorbed into whiskers.

🟣 IGNORE

Outliers removed entirely. Box and whiskers recalculated without them.

📝 Your Exam Question Solved

Question: "Which box plot uses the Hide Outliers option? All three use the same data source."

Box Plot 1 = SHOW
Had gray outlier bins showing extreme values.
Box Plot 2 = HIDE ✅
Clean box + whiskers. Default = Hide.
Box Plot 3 = IGNORE
No box (IQR ≈ $0). Outliers excluded.
✅ Answer: Box Plot 2 (option C). Since Hide Outliers is the default, the normal-looking box plot IS hiding outliers!
❌ "None" was wrong because Box Plot 2 uses the default Hide setting.
🐱 Whiskers = cat's whiskers! They extend from the face (box) to sense how far data reaches before hitting something unusual (an outlier).
