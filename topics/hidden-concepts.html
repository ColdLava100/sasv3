🙈 The 3 "Hidden" Concepts — Data Role vs Page vs Column ▶

SAS VA has 3 different features called "Hidden" — they sound the same but are completely different. This page helps you tell them apart.

📋 Side-by-Side Comparison

🔲 1. Hidden Data Role

"Enables you to include a data item in the query results when you want to use its values but do not want the data item displayed."

🎯 Used for: color-mapped display rules, external links, mapping data sources
📍 Found in: Roles pane of any object (bar chart, table, geo map, etc.)
⚠️ Not available for all objects
⚠️ Must not increase row count in query
⚠️ Cannot assign to Hidden AND Data tip values simultaneously

Example: URL column in Hidden → rows clickable but URL text invisible

📄 2. Hidden Page

"Visible to report designers as a page tab, but not to report viewers."

🎯 Used for: under construction, revision history, notes, reference details
📍 Found in: Page tabs area of a report
✅ Can be source of links (report, page, external URL)
✅ If a link targets a hidden page → viewer sees it like a regular page
❌ Not included in PDF export

Example: A "Data Dictionary" page with column definitions — hidden from viewers but accessible via a "📖 View Data Dictionary" link

🗂️ 3. Hide Column

"To hide a column in a list table, you can right-click the column and select Hide column or use the Roles pane to move the column to the Hidden data role."

🎯 This is simply a shortcut to the Hidden Data Role
📍 Found in: Right-click menu on a list table column
✅ Same functionality as dragging to Hidden role
⚠️ Hidden columns might increase the number of rows for the query

Note: Hide Column is NOT a separate concept — it's just a quicker way to use the Hidden Data Role!

🔍 Hidden Data Role vs Data Tip Values — The Difference

🎯 Key Exam Points:
• Hidden Data Role = data item in query but NOT displayed ⚠️
• Hidden Page = page tab invisible to viewers 📄
• Hide Column = shortcut to Hidden Data Role 🗂️
• Hidden pages are NOT included in PDF ❌
• Hidden pages CAN be accessed via links ✅
• Hidden Data Role cannot be used with Data Tip Values (mutually exclusive) ❌
• Hidden Data Role must not increase row count ⚠️
🧠 Memory Trick:
Role hides a Row/data item 🎭
Page hides the whole Page tab 📄
Column = Click to hide (same as Role) 🗂️

"Role hides data, Page hides tab, Column is just a shortcut lab."
