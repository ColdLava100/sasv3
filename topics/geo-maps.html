⚠️ Rule: "Geography data items created using latitude and longitude in data cannot be used in geo region maps."

🎯 Your Exam Question

Question: "Which is required to create a geo line map?"

❌ Your answer: Latitude and longitude values in the data
✅ Correct: Geographic data provider + Geographic name/code lookup
Why? Geo Line maps need polylines (routes/paths). Lat/Lon gives points only, not lines. You need a geographic data provider (custom routes) or name/code lookup (standard routes).

🧠 Memory Trick

Points = Lat/Lon ✅
Lines = Lookup or Provider ✅
Regions = Lookup or Provider ✅

"Lat/Lon can only give you dots. For lines and areas, you need lookup or provider."
