🔴 WEAKNESS #5: Graph Builder (Roles & Overlay)

Questions you got wrong: "True about roles?", "Where to rename roles?", "Overlay shared role"
Your wrong answers: "You can assign data items" ❌ | Options menu ❌ | Auto share x-axis ❌
✅ Graph Builder rules:
1. Each object has certain required roles. You can add optional roles too.
2. Roles are renamed in the Roles menu (three dots ⋮ next to the role name)
3. For overlay (bar + line chart together), use Roles → Shared Role so both share x-axis but have different y-axes
4. "You can assign data items" is too vague — the specific truth is about required/optional roles

🔴 WEAKNESS #7: Sharing, Prompts & Links

Questions you got wrong: "Set prompt values → which area?", "Export PDF — what's true?", "Dynamic text — what can't be displayed?", "Outline pane"
Your wrong answers: Both areas ❌ | Can't expand clipped ❌ | Interactive filters ❌ | Overview Pane ❌
✅ Set prompt values of target REPORT → Report prompt area ONLY
Set prompt values of target PAGE → Page prompt area ONLY
These options work at the same level they're applied. Report link = report prompts only. Never both.

✅ PDF Export: Filters ARE displayed in appendix. You CAN expand clipped content. ❌ NOT included: Data-Driven Content, Web Content, Hidden pages, Report prompts, Page prompts, Prompt containers.

✅ Dynamic Text in Text Object: Can display: table modified time, interactive filters, measures, parameters. ❌ Cannot display: Category directly (only via expression-based parameter with rank).

✅ Outline Pane: Shows report structure — pages, prompts, and objects. Use to understand report design, rename objects, drag to reorganize.
