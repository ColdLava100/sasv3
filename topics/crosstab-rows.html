📋 Crosstab Rows Role — What Can Go There? RULES ▶

"Both the Columns and Rows roles accept either a single hierarchy or any number of categories."

Data Item TypeCan go in Rows?Why?
📂 Category✅ YesAny number of categories (Product, Region, etc.)
📁 Single Hierarchy✅ YesOne drill-down path (Year > Quarter > Month)
📁📁 Multiple Hierarchies❌ NoOnly ONE hierarchy allowed at a time
📊 Measure❌ NoMeasures go in the Measures role (cells)

🧠 What is a Single Hierarchy?

A drill-down path: Year → Quarter → Month

❌ Why not Multiple Hierarchies?

Rows accepts only one hierarchy at a time.

💡 Memory Trick:
Categories → ✅ Any number
Hierarchies → ✅ One only (Single)
Multiple hierarchies → ❌ No
Measures → ❌ No (go in cells)
