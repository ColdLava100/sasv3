🎯 Auto Chart: Data Items → Chart Type ▶

📋 The Core Rule: Count Your Data Items & Their Types

Data Items You DragChart Type🎯
1 Measure (e.g., Sales)Histogram📊
1 Category (e.g., Product)Bar chart📊
1 Category + 1 MeasureBar chart📊
1 Date/Datetime + Measure(s)Time series plot📈
2 Measures (e.g., Sales, Profit)Scatter plot🔵
3 MeasuresBubble plot🫧
1 Geo + MeasureGeo map🗺️
Category + Category + MeasureGrouped/Stacked bar📊
Category + Multiple MeasuresBar chart📊
🧠 The "1-2-3" Memory System:

1 thing → Histogram (if measure) or Bar (if category)
2 measures → Scatter 🔵
3 measures → Bubble 🫧

Category + Measure → Bar 📊 (always! The default pairing)
Date + Measure → Time Series 📈 (time data = line over time)
Geo + Measure → Map 🗺️ (location data = map)

💡 The Decision Tree

  What data items did you drag?
  │
  ├─ Only 1 item?
  │   ├─ Is it a Measure?      → 📊 Histogram
  │   └─ Is it a Category?     → 📊 Bar chart
  │
  ├─ 2 items?
  │   ├─ Both Measures?        → 🔵 Scatter plot
  │   ├─ Category + Measure?   → 📊 Bar chart
  │   └─ Date + Measure?       → 📈 Time series
  │
  ├─ 3 items?
  │   ├─ 3 Measures?           → 🫧 Bubble plot
  │   └─ 2 Cats + 1 Measure?   → 📊 Grouped Bar
  │
  └─ Has Geography?            → 🗺️ Geo map
