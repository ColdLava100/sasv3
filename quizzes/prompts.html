🎛️ Prompts Quiz ▶

🎛️ Prompts

Q1: What type of filter are Prompts?

✅ Dynamic filters — "Dynamic filters are filters that can be modified or used by the report viewer." Prompts are dynamic; static filters are set by the designer.

Q2 🔥: What is the scope of a REPORT prompt?

✅ All objects, including page prompts — "Report prompts filter all objects in the report, including page prompts." This is a key exam point!
❌ Current page only: That's a PAGE prompt.

Q3: What happens when multiple controls are in a prompt area?

✅ AND operator — "If multiple controls are added, the filters are combined with an AND operator." All conditions must match.

Q4: What is the benefit of a Prompt Container?

✅ Groups + delays Apply — "Prompt containers enable you to minimize space by grouping controls. The execution of actions is delayed until you click Apply."

Q5: Which control is best for selecting a RANGE of values?

✅ Slider — Sliders can select a single value or range, and work with measure and date data items. Dropdown/list are for categories.

Q6: When does an Auto Control choose the best control object?

✅ When you drag a data item to the prompt area — "Auto controls can be created by dragging data items to the report or page prompt area." SAS chooses based on type + cardinality.
❌ Hierarchy: It creates cascading controls, but auto control works for any data item.

Q7: What is a best practice when using controls?

✅ Add instructions as the title — "When using controls, it's a best practice to add instructions as the title of the control object." So viewers know what to select.

Q8 🧠: "Static = _____. Prompts = _____. Report prompt = _____. Page prompt = _____."

✅ Static=Designer, Prompts=Viewer, Report=All pages, Page=One page — The key distinction!
