🎯 M2 L5: Display Rules & Parameters

Q1: Which parameter type is determined by a formula, NOT user input?

✅ Expression-based — "Expression-based parameters are not affected by filters or actions and are based on the entire data source."

Q2: What happens to Expression-Based parameters when filters are applied?

✅ NOT affected by filters — "Because they are not affected by filters or actions, their value remains constant."

Q3: Which display rule type uses Boolean expressions based on a measure?

✅ Expression — "An expression display rule is based on a measure data item Boolean expression; it accepts numeric parameters."

Q4: Parameters can be used in all EXCEPT:

❌ Data source filters — Parameters CAN be used in calculations, filters, display rules, ranks, and URLs.
