🗺️ Geo Maps — Practice Quiz QUIZ ▶

🗺️ Topic 10: Geo Maps — Basic

Q1: A Geo Bubble map uses which geometry type?

✅ Points ● — Geo Bubble, Geo Coordinate, and Geo Pie are all point maps.
❌ Polylines ═══: Used for Geo Line, Geo Network, Geo Line-Coordinate (paths/routes).
❌ Polygons █: Used for Geo Region, Geo Cluster, Geo Contour (filled regions).

Q2: A Geo Line map uses which geometry type?

✅ Polylines ═══ — Geo Line and Geo Network display paths/routes.
❌ Points ●: Used for Geo Bubble, Geo Coordinate, Geo Pie (pinpointing locations).
❌ Polygons █: Used for Geo Region maps (filling regions with color).

Q3: A Geo Region map uses which geometry type?

✅ Polygons █ — Geo Region fills regions with color (choropleth map).
❌ Points ●: Can't show regions — only dots at specific coordinates.
❌ Polylines ═══: Can't fill areas — only draw paths between points.

Q4: What are the 3 ways to create a geography data item?

✅ Geographic name/code lookup, Latitude & Longitude in data, Geographic data provider
❌ Measure/Category/Parameter: These are data item types, not methods for creating geography data items.

Q5: Which type of geo map is NOT a point map?

✅ Geo Line — It's a polyline map (paths/routes).
❌ Geo Bubble: Point map — bubbles at coordinates, sized by a measure.
❌ Geo Coordinate: Point map — dots at coordinates.
❌ Geo Pie: Point map — mini pie charts at coordinates.

Q6: Which map type combines polygons + points in one view?

✅ Geo Region-Coordinate — Fills regions AND shows coordinate dots.
❌ Geo Region: Polygons only — no coordinate dots.
❌ Geo Cluster: Groups regions — doesn't show individual points.
❌ Geo Bubble: Points only — no polygon regions.

🗺️ Topic 10: Geo Maps — Advanced

Q7 🔥: Sales data across US states with uneven distribution — California has $50M but Delaware has $2M. Which point map is BEST?

✅ Geo Bubble — From course: "Geo bubble maps are useful when data is not evenly distributed." Bubble size represents the measure, so Delaware doesn't look as big as California.
❌ Geo Coordinate: Should only be used with evenly distributed data. Uniform dots would make tiny and big states look equal.
❌ Geo Pie: Used for showing distribution among a few groups per location, not for comparing a single measure across many regions.

Q8 🔥: Display flight routes between airports. The route path matters AND you want airport names on hover. Which type?

✅ Geo Line-Coordinate — Shows route paths (polylines) AND coordinate markers at endpoints with hover info.
❌ Geo Line: Shows paths but no interactive dots or hover labels on endpoints.
❌ Geo Network: For interconnected nodes with many-to-many connections, not simple routes.
❌ Geo Bubble: Bubbles show measures at points — can't show route paths.

Q9 🔥: Data has store addresses as Lat/Lon. You want regions colored by total sales. What can you use?

✅ Both B and C — You CANNOT use Geo Region with Lat/Lon!
❌ Geo Region: Course says: "Geography data items created using latitude and longitude in data cannot be used in geo region maps."
✅ Geo Coordinate (dots colored by sales) and Geo Bubble (bubbles sized by sales) both work with Lat/Lon data items.

Q10 🔥: A Geo Network shows connections between nodes. What must be TRUE about source & target data items?

✅ Both must be geographic data items — Course: "If both the source and target nodes are geographic data items, then the network can be overlaid on a geo map."
❌ Both measures: Networks need geographic locations, not numeric values.
❌ Different data sources: They can be from the same source — what matters is geographic classification.

Q11 🔥: What permissions are needed to create a geographic data provider? Why does this matter for geo line maps?

✅ Special user permissions required — Course: "Special user permissions are required to define and edit geographic data providers." Geo line maps need polylines, which often come from a geographic data provider (custom routes). Without permission, you can't create custom polyline data!

Q12 🔥: Show profit by country (colored regions) + store locations (dots) in one map. Which type?

✅ Geo Region-Coordinate — Combines polygon regions (colored by profit) WITH coordinate dots (store locations) in a single map.
❌ Geo Region: Shows regions but NO coordinate dots.
❌ Geo Bubble: Shows points but NO filled regions.
❌ Two separate maps: Possible but unnecessary when Geo Region-Coordinate exists!

Q13 🔥: Sales territory groupings — custom clusters not matching state lines. Which combination?

✅ Geo Cluster + Geographic data provider — Course: "To use custom polygonal shapes, a geographic data provider must be defined." Standard lookups won't work for custom territories.
❌ Geo Region + Lookup: Lookup only knows standard boundaries (countries, states), not custom clusters.
❌ Geo Bubble + Lat/Lon: Shows points, not region groupings.

Q14 🔥: A Geo Contour map is BEST for what?

✅ Density patterns — Contour lines show areas of equal value/density (like elevation or heat maps).
❌ Exact sales: Use Geo Region (colored states/countries) for exact values per region.
❌ Store locations: Use Geo Coordinate or Geo Bubble for individual points.
❌ Airline routes: Use Geo Line or Geo Line-Coordinate for paths between locations.

Q15 🔥: Identify the FALSE statement about geo maps:

✅ FALSE: Lat/Lon works with ALL types — This is wrong! Lat/Lon CANNOT be used with Geo Region maps.
✅ Geo Bubble + uneven data: TRUE — from course: "Geo bubble maps are useful when data is not evenly distributed."
✅ Geo Coordinate + even data: TRUE — from course: "Geo coordinate maps should be used only when there is an even distribution of values."
✅ Provider from ESRI/CAS: TRUE — from course: "The geographic data provider can be created from either an Esri feature service or a CAS table."

Q16 🏆: Supply chain dashboard: distribution centers as bubbles, delivery routes, sales regions colored. Minimum geo maps needed?

✅ 1 — custom geo map layers all three — Course: "You can mix and match these elements to create your custom geo map." A custom geo map in Graph Builder can layer Geo Map (polygons) + Scatter Plot (points) + Bubble Plot (bubbles) + Line Plot (routes) all in one.
