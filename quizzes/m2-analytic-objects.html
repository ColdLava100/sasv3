Q1: What is the default forecast horizon?

✅ 6 periods — Default forecast horizon is 6 periods. Default confidence interval is 95%.

Q2: What is the default confidence interval for forecasting?

✅ 95% — Default confidence interval. The colored band around the forecast line represents this range.

Q3: In Network Analysis, Source values must be:

✅ A subset of Target — For network analysis, Target values must be a subset of Source values. All targets must appear as sources.

Q4: Which What-If Analysis type modifies future values of underlying factors?

✅ Scenario Analysis — Scenario Analysis modifies underlying factor values. Goal-Seeking sets target forecast and finds required factor values.
