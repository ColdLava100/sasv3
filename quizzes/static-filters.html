Practice Quiz — Static Filters

🏷️ Static Filters Quiz ▶

🏷️ Static Filters

Q1: Which filter operates at the REPORT level?

✅ Data Source Filter — "A global filter that subsets data for the entire report." The other three are object-level filters.

Q2: A Basic Filter uses how many data items?

✅ 1 data item — "A basic filter includes only one data item."
❌ 2+: That's Advanced Filter.
❌ Aggregated: That's Post-Aggregate Filter.

Q3: How many Data Source Filters can you have per data source?

✅ 1 per data source — "There is a limit of one data source filter per data source." Use a complex Boolean expression for multiple conditions.

Q4 🔥: A filter needs to show only products where TOTAL sales > $10,000 (after grouping). Which filter?

✅ Post-Aggregate Filter — "Subsets using aggregated values, not individual values." After SUM(Sales) is calculated, it keeps only products > $10K.
❌ Basic/Advanced: Work on detail data (individual rows), not totals.

Q5: What types of filters can be converted to a Common Filter?

✅ Basic, Advanced, OR Post-Aggregate — "A common filter can be a basic, advanced, or post-aggregate filter that is shared between objects."

Q6: Who defines static filters?

✅ Report designer — "Static filters must be defined and applied by the report designer." (Dynamic filters like prompts are used by viewers.)

Q7 🔥: Which is TRUE about a Data Source Filter?

✅ Updates cardinalities — "Data pane cardinalities are updated" when a Data Source Filter is applied. It acts as a pre-filter.
❌ After object-level: It's applied BEFORE (acts as pre-filter).
❌ Convert to Common: Data Source cannot be converted — only Basic/Adv/Post can.
❌ Multiple: Only 1 per data source.

Q8: The Detail data option is disabled if what exists?

✅ A Post-Aggregate Filter — "The Detail data option is not available if the list table already includes a post-aggregate filter."
