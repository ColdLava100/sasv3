🎮 Interactive Reports Quiz ▶

🎮 Designing Interactive Reports

Q1 🔥: Can Automatic and Manual actions be combined on the same page?

✅ No — "Each page can either have manual actions or automatic actions, but not both." This is a critical exam point! ❌

Q2: What does Linked Selection do?

✅ Highlights data — "The automatic linked selection action highlights related data in multiple objects." Other rows stay visible but grayed out.
❌ Subsets: That's a filter action.

Q3 🔥: What does the Two-Way Filter action do?

✅ Filters ALL objects both ways — "The automatic two-way filter action applies filters between all objects, even previously selected objects." ⚠️ Don't use with required prompts!

Q4: Which objects CANNOT be the source of a filter action?

✅ Images, Text, Web Content, Job Content — "These additional objects cannot be the source of a filter action: images, job content, text, and web content."
❌ Dropdowns/Bar charts: They CAN be sources.

Q5: Filter actions are ____________ and ____________.

✅ Unidirectional and transitive — "All filter actions are unidirectional and transitive." Source→Target only (uni) and A→B→C means A→C (transitive).

Q6: Which link type navigates to another page within the same report?

✅ Page Link — Page links navigate within the same report. Report links go to a different report. URL links go to a website. Return links go back.

Q7: What happens when you create a page link to a Hidden Page?

✅ Appears like a normal page — "In view mode, a linked hidden page will appear like a regular page." The tab is hidden, but the page is accessible via link.

Q8 🧠: "Prompts = ____. Actions = ____. Links = ____."

✅ Prompts=TV remote (pick), Actions=Steering wheel (respond), Links=GPS (navigate) — Memorize this analogy!
