📤 M2 L7: Sharing & Managing

Q1: Which is NOT included in PDF export?

❌ Data-Driven Content — Data-Driven Content, Web Content, Hidden pages, Report prompts, Page prompts, and Prompt containers are NOT included in PDF.

Q2: Who must enable Guest Access?

✅ Administrator — Guest Access requires admin to enable via SAS Environment Manager.

Q3: What does Copy Embeddable Markup generate?

✅ A code snippet for SDK — Copy Embeddable Markup generates code for SAS Viya SDK for JavaScript.

Q4: Export Report Package creates:

✅ A static snapshot of data + queries — Report Package is self-contained, no Viya needed. Can view offline.
