🎨 M2 L2: Graph Builder

Q1: Where is Graph Builder accessed from?

✅ Applications menu — "Graph Builder is accessible from the applications menu. It is available to all users."

Q2: What is the default cell in a Graph Builder layout?

✅ Cell A1 — In Graph Builder, the default starting cell for graph elements is Cell A1 (top-left).

Q3: How many graph elements are available in Graph Builder?

✅ Over 15 — Graph Builder has over 15 graph elements available including Bar, Line, Scatter, Pie, etc.
