Q1: Which operation is NOT available in Visual Analytics?

❌ Append — Append (adding rows) is NOT available in Visual Analytics. Joins (adding columns) are available.

Q2: What does a Join do?

✅ Adds columns by matching rows — A data join combines two tables by matching data rows and adding columns.

Q3: Which function extracts a substring?

✅ Substr — Substr(string, start, length) extracts a substring. Trim removes spaces, Upcase changes case, Length gets string length.

Q4: Which function counts intervals between two dates?

✅ IntCK — IntCK counts intervals between dates. IntNX advances a date by intervals.

Q5: Aggregated functions evaluate on:

✅ Aggregated data — Aggregated functions evaluate on aggregated data, not detail data.
