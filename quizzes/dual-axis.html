📊 Dual Axis Charts Quiz ▶

📊 Dual Axis Charts

Q1: Which category dependency does a Dual Axis Bar Chart require?

✅ Independent categories — "Use a dual axis bar chart when the categories are independent." Values for one category don't affect another.
❌ Dependent: That's for Line charts.

Q2: In a Dual Axis Bar-Line Chart, which axis starts at zero?

✅ Bar axis only — "Axis for bar starts at zero. Axis for line does not start at zero."

Q3 🔥: What is NOT added to the Dual Axis Time Series Plot?

✅ Forecasts — "Forecasts are not added to the dual axis time series plot." This is a key exam distinction vs regular Time Series Plot.
❌ Lattice: Bar chart can use lattice rows.

Q4: Which dual axis chart type requires a date/datetime data item?

✅ Dual Axis Time Series Plot — It requires a Time axis (date/datetime). The other three use a Category axis.

Q5 🔥: A report shows facility efficiency and number of employees by month. Values in February are impacted by January. Which dual axis chart?

✅ Dual Axis Line Chart — The categories are DEPENDENT (each month's value affects the next), so use Line-Line.
❌ Bar: Requires independent categories.
❌ Time Series: Requires date/datetime, but month names as categories are ordinal.

Q6: Where are Dual Axis Charts located in the Objects pane?

✅ Graphs group — "Dual axis charts are located in the Graphs group in the Objects pane."
❌ Content: That's for Text, Image, Web Content, etc.

Q7: Which axis represents the PRIMARY measure?

✅ Left Y-axis — "Use the left Y axis to show the primary measure and the right Y axis for the secondary measure."

Q8 🧠: "Bar-Bar = _____. Bar-Line = _____. Line-Line = _____. Time Series = _____."

✅ Indep / Volume+Rate / Dependent / Seasonal — Bar-Bar = Independent categories, Bar-Line = Volume bars + Rate line, Line-Line = Dependent categories, Time Series = Seasonal patterns.
