📈 Time Variant Charts Quiz ▶

📈 Time Variant Charts

Q1: Which Time Variant chart requires exactly 4 measures?

✅ Vector Plot — Requires 4 measures: X Origin, X, Y Origin, Y.
❌ Bubble Change: Requires 6 measures!
❌ Waterfall: Only needs 1 measure.
❌ Schedule: Needs Task + Start + Finish (no measures).

Q2: Which chart requires the most data items (6 measures)?

✅ Bubble Change Plot — 6 measures: X start, X end, Y start, Y end, Size start, Size end.

Q3: What does a Waterfall chart show?

✅ Incremental changes → final value — A progressive bar chart showing cumulative effects.

Q4: A Step Plot is sorted by what, by default?

✅ Ascending by X axis — "By default, the step plot is sorted in ascending order by the value of the X axis."
❌ Descending: That's the default for Dual Axis Bar charts.

Q5: What roles does a Schedule Chart require?

✅ Task + Start + Finish — Schedule charts (Gantt) need task name, start date, and finish date. No measures needed!

Q6 🔥: Which chart is also called a "progressive bar chart", "Mario chart", or "bridge chart"?

✅ Waterfall Chart — "Also known as: progressive bar charts, flying bricks charts, Mario charts, and bridges (in finance)."

Q7: Where are Time Variant Charts located?

✅ Graphs group — Same as Dual Axis Charts. Content group has Text, Image, Web Content, etc.

Q8 🧠: Match: "Step = ___. Vector = ___. Bubble = ___. Waterfall = ___. Schedule = ___."

✅ Step=Staircase, Vector=4 Measures, Bubble=6 Measures, Waterfall=Incremental, Schedule=Gantt
