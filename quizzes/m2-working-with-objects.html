Q1: Bubble plots display data how by default?

✅ Non-aggregated — "By default, bubble plots display non-aggregated data, attempting to display one bubble per row." Assign Group first for large data.

Q2: What is the max number of measures for a Correlation Matrix?

✅ 60 — "The maximum number of measures that can be assigned to the correlation matrix is 60."

Q3: Correlation matrix uses which coefficient?

✅ Pearson's — "The correlation values are calculated by using Pearson's correlation coefficient."

Q4: A correlation of |r| = 0.4 is considered:

✅ Moderate — Weak ≤0.3, Moderate 0.3-0.6, Strong ≥0.6. |r|=0.4 falls in moderate range.

Q5: What 3 components make up the Automated Explanation object?

✅ Natural language + importance bar chart + relationship plot — The 3 components of Automated Explanation.

Q6: Which roles CANNOT be assigned together on a bubble plot?

❌ Group and Color — "You cannot assign both the Group and Color role at the same time."
